# Sparkamp Window Requirements

## 1. Main Window / Player View

The primary interface for playback control and track display.

### Shared Features
- Display current track info (title, artist, duration)
- Transport controls: Previous, Play/Pause, Stop, Next
- Seek functionality (seekbar/slider)
- Volume control (0-100%)
- Repeat modes: Off -> Song -> Playlist
- Shuffle toggle
- Now-playing display with elapsed/remaining time
- Track index indicator (e.g., "3 / 15")

### TUI Specific
- Mini visualizer (22 columns, animated bars)
- Marquee scrolling title for long names
- Progress gauge (text-based bar)
- Transport hints at bottom
- Scrollable playlist panel
- Playlist toggle visibility

### GUI Specific
- GTK4-based main window
- Picture widget for album art (clickable for larger view)
- Horizontal seekbar
- Toggle buttons with icons for repeat/shuffle/EQ/ML
- Volume slider
- Status bar with GNOME integration
- GNOME logo click -> Settings, right-click -> theme toggle

---

## 2. Playlist Window

The track queue/list view.

### Shared Features
- Display all queued tracks
- Show track number, title, artist, duration
- Play track on selection
- Remove tracks from list
- Clear all tracks
- Track count display

### TUI Specific
- Collapsible panel in main view
- Duration column
- 1-based track numbering

### GUI Specific
- Separate GTK4 window
- Multi-select ListBox
- Drag-and-drop reordering
- External file drag-and-drop support
- Add Files button (opens native FileDialog)
- Add Folder button
- Remove selected button
- Clear All button

---

## 3. Jump to Track / Search

Quick track search and navigation.

### Shared Features
- Text search input
- Live filtering as user types
- Case-insensitive search
- Results show title + artist
- Keyboard navigation (up/down arrows)
- Enter to play selected
- Escape to close

### TUI Specific
- Yellow highlight for selected result
- Auto-selects first result on typing

### GUI Specific
- GTK4 SearchEntry widget
- ListBox for results
- Window appears as dialog/overlay

---

## 4. Settings / Preferences

Configuration interface.

### Shared Features
- Appearance settings (theme, skins)
- Behavior settings (autoplay, etc.)
- Media library folder configuration
- Plugin directory configuration

### TUI Specific
- 5-tab overlay interface
- Inline text editing
- Arrow key navigation
- Save on close

### GUI Specific
- 5-tab Notebook widget
- Theme dropdown selector
- Custom skin path entry
- Watched folders list with add/remove
- Native GTK4 dialog

---

## 5. Equalizer

Audio frequency adjustment.

### Shared Features
- 10-band equalizer (32Hz to 16kHz)
- Pre-amp control (50%-150%)
- Enable/disable toggle
- Preset management (load/save)
- Band range: -12 to +12 dB
- Reset to flat

### TUI Specific
- 11-column layout (pre-amp + 10 bands)
- Left/Right arrows: navigate bands
- Up/Down arrows: adjust gain
- PgUp/PgDn: larger adjustments
- Band number display (0-9 + P for pre-amp)

### GUI Specific
- Horizontal sliders for each band
- Pre-amp as separate horizontal scale with marks
- Preset dropdown
- Enable checkbox
- Reset button
- Real-time engine updates
- Config saved on close

---
