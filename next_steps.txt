## Phase 4: Zip-Based Skin Packaging (Future)
Purpose: Distribute skins as single files (like Winamp .wsz files)

### Implementation Ideas
1. Rename .zip to .sparkamp-skin
2. Structure:
   ```
   myskin.sparkamp-skin (zip archive)
   ├── skin.css
   ├── images/
   │   ├── play.png
   │   ├── play_active.png
   │   └── background.png
   └── manifest.toml (optional)
       name = "My Skin"
       author = "..."
       version = "1.0"
   ```

3. File operations:
   - Extract to ~/.config/sparkamp/skins/myskin/
   - OR: Load directly from zip without extraction
   - Registry: ~/.config/sparkamp/known_skins.toml

### Benefits
- Single file distribution
- Integrity verification (checksum)
- Easy uninstall (delete extracted folder)
